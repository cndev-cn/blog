Zend_studio-11.0.1破解步骤/在线汉化/在线安装主题等相关说明：

[1]破解步骤

[1.1]破解补丁的安装

解压文件[Zend11_keygen.tar.gz]，复制[com.zend.verifier_11.0.1.v20140815-1236.jar]到Zend_Studio的安装目录下的plugins目录，并覆盖同名文件。

[1.2]生成可用的激活码
此步骤需要你的电脑JAVA环境，因为需要运行注册机[keygen.jar]来生成与破解补丁相匹配的激活码。（如果你电脑没用JAVA环境又懒得安装的话，在本文后面我会提供一些生成的可用的激活码，你直接复制即可。）

```
# 检查下自己电脑是否有JAVA环境
phpha@asus:~$ java -version
java version "1.7.0_65"
OpenJDK Runtime Environment (IcedTea 2.5.1) (7u65-2.5.1-4ubuntu1~0.14.04.2)
OpenJDK 64-Bit Server VM (build 24.65-b04, mixed mode)
# 运行注册机[keygen.jar]
phpha@asus:/home/Zend11_keygen$ ls
com.zend.verifier_11.0.1.v20140815-1236.jar  keygen.jar
phpha@asus:/home/Zend11_keygen$ java -jar keygen.jar
```

执行以上命令后，会弹出注册机的窗口，自己随意生成吧。

[2]汉化步骤

运行Zend_studio，然后[Help]->[Install New Software]，然后新添加个在线的汉化地址：

http://download.eclipse.org/technology/babel/update-site/R0.12.0/juno/

离线汉化包：http://eclipse.org/downloads/download.php?file=/technology/babel/update-site/R0.11.1/babel-R0.11.1-juno.zip
