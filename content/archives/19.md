首先需要说明的是，memcache与memcached的区别，memcache是客户端的扩展，memcached是服务端。


<!--more-->


1、安装libmemcached库
```bash
[root@PHPHa ~]# yum install libmemcached
```
2、PHP安装Memcache扩展
```bash
[root@PHPHa ~]# wget -c http://pecl.php.net/get/memcache-3.0.8.tgz
[root@PHPHa ~]# tar -zxvf memcache-3.0.8.tgz
[root@PHPHa ~]# cd memcache-3.0.8
[root@PHPHa ~]# /usr/local/php/bin/phpize
[root@PHPHa ~]# ./configure  --with-php-config=/usr/local/php/bin/php-config --enable-memcache --with-zlib-dir
[root@PHPHa ~]# make && make install
/usr/local/php/lib/php/extensions/no-debug-non-zts-20090626/memcache.so
[root@PHPHa ~]# vim /usr/local/php/etc/php.ini
extension = /usr/local/php/lib/php/extensions/no-debug-non-zts-20090626/memcache.so
session.save_handler = memcache
session.save_path = "tcp://127.0.0.100:11211"
[root@PHPHa ~]# service php-fpm reload
```
补充：本文介绍的是PHP的Memcache扩展的安装，也就是说客户端的安装。安装后发现一个问题，SESSION丢失，想到了安装Memcached服务端的时候，防火墙没有添加11211端口，修改防火墙规则后，一切正常。
