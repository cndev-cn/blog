本文介绍下开启MySQL远程连接权限的相关操作，需要说明的是这样做存在安全隐患，建议把IP设置成固定的IP，千万别用%来匹配所有的IP。


<!--more-->


 1. 开启远程连接权限

```mysql
-- 开启IP为8.8.8.8的主机对数据库test的所有表的远程连接及所有操作权限并设置密码为123456
mysql> GRANT ALL PRIVILEGES ON test.* TO root@"8.8.8.8" IDENTIFIED BY "123456";
-- 刷新数据库权限
mysql> flush privileges;
```

 2. 开启防火墙

```bash
# 开启3306端口访问权限
[root@PHPHa ~]# iptables -I INPUT -p tcp --dport 3306 -j ACCEPT
# 保存防火墙规则
[root@PHPHa ~]# service iptables save
```
